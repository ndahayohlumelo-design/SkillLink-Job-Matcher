# SkillLink-Job-Matcher
App

---

## **Features**

- Enter your name to personalize results  
- Select multiple skills from a modern, interactive checklist  
- Receive job suggestions dynamically based on the selected skills  
- Duplicate job prevention  
- Smooth UI with hover effects, animations, and responsive design  
- Fully responsive for mobile and desktop  

---

## **Technologies Used**

- **HTML5** – structure of the app  
- **CSS3** – modern styling, gradients, and responsive design  
- **JavaScript (ES6)** – interactive functionality and logic
